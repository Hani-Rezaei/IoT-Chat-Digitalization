Detailed Description

Header-Datei für die SAUL-Geräteverwaltung und LED-Steuerung.

Diese Datei enthält die Deklarationen für die Steuerung von LED-Geräten, die mit dem SAUL-Register (Sensor, Actuator, and User Logic) verwaltet werden. Sie ermöglicht das Ein- und Ausschalten von LEDs über ihre Gerätenamen.

Setzt den Zustand einer LED basierend auf ihrem Gerätenamen.

Diese Funktion sucht ein Gerät anhand des übergebenen Namens im SAUL-Register und setzt dessen Zustand auf AN oder AUS, abhängig vom Parameter is_on.

Parameters
[in]device_nameName des LED-Geräts im SAUL-Register.
[in]is_onStatus der LED: 1 für AN, 0 für AUS.
Returns
0 bei Erfolg, -1 wenn das Gerät nicht gefunden wurde, -2 falls das Setzen des Zustands fehlschlägt.
